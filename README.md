Privacy Policy
DevTrack Attachment Preview Extension

Overview
This extension is intended for use within the DevTrack enterprise system environment.

Data Collected
When a user previews or batch downloads attachments, the extension collects the following operation metadata:

Operation time

Operator name (as provided by the DevTrack system)

Attachment file name

The extension does not collect browsing history, cookies, or data from unrelated websites.

Purpose of Data Collection
The collected metadata is transmitted to an internal enterprise service endpoint for auditing and usage tracking purposes only.

Data Storage
The extension does not permanently store personal data locally. Any temporary data processing is limited to functional requirements.

Data Sharing
The collected data is not sold, shared with third parties, or used for advertising purposes.

Scope Limitation
The extension operates only on declared DevTrack domains and does not access data from other websites.
